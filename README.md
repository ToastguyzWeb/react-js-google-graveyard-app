# 🪦 Google Graveyard - React App
A comprehensive showcase of **discontinued Google projects** and services. Built with **React** + **Vite**, **React Router DOM**, and **TailwindCSS**.
<br/>
Explore the extensive history of Google's discontinued products and services through an interactive and user-friendly interface. Users can **browse, search, and filter** through hundreds of Google projects that have been shut down over the years. The app provides detailed information about each project including launch dates, shutdown dates, descriptions, and categories.
<br/>

## 🌟 Features
- 📱 **Fully Responsive** design across all devices
- 🔍 **Advanced Search** functionality to find specific projects
- 🏷️ **Category Filtering** - Filter by project types and categories
- 🧭 **Multi-page Navigation** with React Router DOM
- 📄 **Legal Pages** - About Us, Terms of Service, Privacy Policy
- ⚡ **Lightning Fast** performance with optimized loading
- 🎨 **Modern UI/UX** with smooth animations and transitions

## 🛠️ Setup Instructions

1. **Clone this repository:**
   ```bash
   git clone https://github.com/ToastguyzWeb/react-js-google-graveyard-app.git
   cd react-js-google-graveyard-app
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```

- **Real-time search** through project names and descriptions
